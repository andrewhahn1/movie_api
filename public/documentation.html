This API (myFlix) will provide access information about different movies. This project will including the server, business logic,and business layers of the application. It will consist of a well-designed REST API and architected database built using JavaScript, Node.js, Express, and MongoDB.

URL Endpoints and Request Methods

Requet URL HTTP Method Response
Create new users /users POST JSON Object including:
{
id:
name:
favoriteMovies:
}
Add movie to favories /users/:id/:movieTitle POST A message indicating whether the movie was successfully added from the user's favorited movies
Return list of all movies /movies GET An array of JSON objects holding data about all the movies in the database
Return data about a single movie by title /movies/:title GET A JSON object holding data about the requested movie:
{
"Title": " ",
"Description": " ",
"Genre": " ",
"Director": " ",
}
Return data about a genre /movies/genre/:genreName GET Return an object of the genre and definition
Return data about a director /movies/directors/:directorName GET Return a JSON object about the requested director:
{
"Name": " ",
"Bio": " ",
"Birth": " ",
}
Update user info /users/:id PUT Returns a message that user info has been udpated
Remove movie from favorites /users/:id/:movieTitle DELETE Returns message that a movie was removed from favorites
Allow existing user to deregister /users/:id DELETE Returns message that user was deleted
